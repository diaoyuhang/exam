1. 某资料借阅中心采用 RFID 对使用后的资料进行排序，读取需要定位排序所在的书架的资料后，采用下述排序算法对打乱后的资料进行排序，`sort` 函数如下：

```
void sort(int original[], int result[], int location[], int n)
{
    int i,j;
    for(i=0;i<n;i++) location[i]=0;
    for(i=0;i<n-1;i++)
        for(j=i+1;j<n;j++)
            if(original[i]<=original[j]) location[j]++;
            else location[i]++;
    for(i=0;i<n;i++) result[location[i]]=original[i];
}
```

其中，数组 `original` 存放资料的数字编号，数组元素的顺序是当前资料乱序的排列顺序；数组 `result` 和 `location` 存放使用 `sort()` 函数排列的结果。

------

### (1)给定数据初始化语句为：

```

result[5], location[5];
```

------

### (2) 分析数组 `location` 的内容对管理员整理资料顺序的作用

管理员可以根据 `location` 数组中的内容，按照相应的位置索引，有序地整理资料。通过这种方式，管理员可以迅速找到所需的资料，提高了整理和检索效率。这样的排序方法通常有助于图书馆或资料借阅中心等场所更有效地管理和利用资料。
