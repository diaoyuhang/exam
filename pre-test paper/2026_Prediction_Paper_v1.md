# 2026 Computer Industry Examination Prediction Paper

## 一、考试说明

- 试卷名称：2026 年计算机行业考试预测卷
- 考试时间：120 分钟
- 试卷总分：60 分
- 注意事项：
  1. 本试卷分为单项选择题、多项选择题、简答题、实务题四个部分。
  2. 单项选择题只有 1 个正确答案，多项选择题有 2 个或以上正确答案，错选不得分，漏选酌情给分。
  3. 请结合题意作答，简答题与实务题应条理清晰、术语准确。
  4. 本预测卷在保持历年试卷风格的基础上，适度融入人工智能、云计算、数据工程与网络安全等近年高频热点。

## 五、实务题（共 3 题，共 40 分）

### 1. 智能知识服务平台设计题（12 分）

某职业培训机构拟建设一个“智能知识服务平台”。学员可以上传学习资料，系统对资料进行分类、切片、向量化存储；学员提问时，系统先检索相关知识片段，再调用大语言模型生成答案；管理员可对高风险回答进行复核，并记录问答日志。

请完成下列任务：

1. 分析该平台的核心实体，并给出主要实体之间的关系。
2. 设计该平台的关系模式，至少包含学员、资料、知识片段、问答记录、管理员等核心数据对象。
3. 简述“检索增强生成”相比“模型直接作答”的主要优势。

### 2. 企业业务系统架构题（14 分）

某连锁零售企业原有系统采用单体架构，现因线上业务快速增长，准备升级为基于云平台的分布式业务系统。企业要求新系统具备以下特点：

- 订单、库存、会员、支付等模块可独立开发和部署。
- 高峰期可自动扩容。
- 不同门店访问总部系统时应保证数据安全。
- 关键业务故障时应尽可能降低影响范围。

请完成下列任务：

1. 说明该企业从单体架构迁移到微服务架构的主要原因。
2. 结合题意，写出实现上述目标可采用的 4 项关键技术或机制。
3. 说明在分布式系统中，为什么需要统一网关、日志监控与服务治理。

### 3. 数据安全与程序实现题（14 分）

某单位拟开发一个文件上传审查程序。系统要求对上传文件名进行初步检查：如果文件扩展名属于 `pdf`、`docx`、`xlsx`，则允许进入下一步人工审核；否则直接拒绝。现给出伪代码如下：

```text
function checkFileType(filename)
begin
    ext = getExtension(filename)
    if ext == "pdf" or ext == "docx" or ext == "xlsx" then
        return true
    else
        return false
    end if
end
```

请完成下列任务：

1. 简述上述程序的功能，并说明其属于顺序、选择、循环三种基本结构中的哪一种。
2. 该程序仅依据文件扩展名判断是否放行。说明这种做法可能带来的安全风险。
3. 从文件上传安全角度，提出至少 3 条改进措施。
4. 若系统还需记录每次上传操作的“用户编号、文件名、上传时间、审核结果”，请设计一张数据表，并写出主要字段。
